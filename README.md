# 沙畫遊戲 · Peel. Pour. Paint.

iPad 上的沙畫小遊戲。倒沙、傾斜裝置，讓沙子流進隱藏的圖案裡，慢慢「顯現」一顆蘋果。

## 玩法

- **點畫面** → 在該處倒一把沙
- **傾斜 iPad** → 把沙導進圖案（iOS 需點「啟用傾斜」授權）
- 沙黏到哪一層就染上那層顏色（紅果身、綠葉、棕梗、亮面、外框）
- 填滿 **85%** 即過關

桌面測試：沙預設往下掉，可用**方向鍵**改變重力方向。

## 技術

- 單一 HTML 檔，JS 全部 inline，無外部依賴
- 三層 Canvas：背景引導圖 / 已黏沙（持久）/ 移動中沙粒
- 約 300 顆胖沙粒（3–5px），重力模擬 + 12% 黏附機率
- 圖層用 Bezier/橢圓路徑定義，讀 alpha 建立 `Uint8Array` 遮罩
- 程式內部分為 `LayerManager` / `ParticleSystem` / `InputManager` 三個 class
