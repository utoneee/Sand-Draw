# 沙畫遊戲 · Peel. Pour. Paint.

iPad 上的禪意沙畫小遊戲。從畫廊選一幅圖案，撕開貼紙露出黏貼面，倒沙、傾斜裝置，
讓沙子流過黏貼處被黏住，慢慢「顯現」一幅作品。

線上版：https://utoneee.github.io/Sand-Draw/

## 玩法

1. **選圖案** — 畫廊式選單，依分類（冥想／山景／水波／夜空／靜物／水果）挑一幅
2. **撕貼紙** — 點圖案上的區塊，撕開那塊露出黏貼面（刀模線標出可填的分區）
3. **倒沙** — 點顏色，在畫面中央倒一堆沙
4. **傾斜** — 傾斜 iPad（或拖曳盤面）讓沙滑過黏貼面被黏住、填出顏色
5. 填好一塊倒掉散沙，換下一色；全部填滿即完成，可存圖欣賞

桌面測試：沙預設往下落，可用方向鍵改變重力方向。

## 圖案

漣漪、遠山、海浪、月相、茶、花瓶、蘋果、香蕉、愛心。

## 技術

- 單一 HTML 檔，JS 全部 inline，僅用 Google Fonts CDN（沒網路時自動退回系統字）
- 多層 Canvas + 粒子系統，沙粒以鄰近分離堆疊出落地厚度
- 程式內分 `LayerManager`（遮罩／撕膜／進度）、`ParticleSystem`（物理）、`InputManager`（傾斜／輸入）
- 明暗主題、作品存圖

## 加新圖案的三種方式

1. **程式畫**（`shape(c)`）— 幾筆就能畫的簡單形狀，直接用 Canvas 路徑（如蘋果、香蕉）
2. **SVG 向量路徑**（`svgRegion(color, d)`）— 在 Illustrator／Figma／Inkscape 畫好，
   以 `viewBox="0 0 480 480"` 匯出，把每段 path 的 `d` 字串貼進來。向量、清晰、可內嵌單檔。
   **推薦給較複雜的圖案。** 原則：區塊少、形狀粗大，沙子才填得漂亮。
3. **點陣圖遮罩**（`buildMaskPattern`）— 用 PNG 索引圖當題目（需放 `patterns/` 資料夾隨檔部署）。
   保留此基礎設施，給「需要照片質感」的少數情況使用。
