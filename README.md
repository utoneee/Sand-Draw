# 沙畫遊戲 · Peel. Pour. Paint.

iPad / 手機上的禪意沙畫小遊戲。從畫廊選一幅圖案，撕開貼紙露出黏貼面，倒沙、傾斜裝置，
讓沙子流過黏貼處被黏住，慢慢「顯現」一幅作品。完成後自動收進你的收藏。

線上版：https://utoneee.github.io/Sand-Draw/

## 玩法

1. **選圖案** — 畫廊式選單，依分類（冥想／靜物／夜空／天象／植物／水果／水波…）挑一幅
2. **撕貼紙** — 點圖案上的區塊，那塊變黃（選中）；再點一次、或按右側「撕下」鈕，撕開露出黏貼面
3. **倒沙** — 點顏色，在畫面中央倒一堆沙（同一區倒不同色會混出新色）
4. **傾斜** — 傾斜裝置（或用手指拖曳盤面）讓沙滑過黏貼面被黏住、填出顏色
5. 填好一塊按「清沙」倒掉散沙，換下一色；全部填滿即完成，收進收藏可放大欣賞

> 手機族小撇步：覺得沙子流動偏慢，可在「工具⚙ → 沙子速度」切到「快」。

## 操作小提醒

- **遊玩時請鎖定螢幕旋轉**，傾斜手感才穩定。
- 右上可隨手切換**深灰／淨白**主題——主題色會一起決定你存圖的底色。
- 桌面測試：沒有傾斜感測器時，沙預設往下落，可用方向鍵改變重力方向。

## 技術

- 單一 HTML 檔，JS 全部 inline，離線可玩（PWA，僅字型走 Google Fonts CDN，沒網路自動退回系統字）
- 多層 Canvas + 粒子系統，沙粒以鄰近分離堆疊出落地厚度
- 圖案以 SVG 向量路徑定義（`viewBox="0 0 480 480"`），逐塊向量、清晰、可內嵌單檔
- 程式內分 `LayerManager`（遮罩／撕膜／進度）、`ParticleSystem`（物理）、`InputManager`（傾斜／輸入）
- 明暗主題、平板／手機介面切換、作品收藏與存圖

## 加新圖案

在 Illustrator／Figma／Inkscape 畫好圖，以 `viewBox="0 0 480 480"` 匯出 SVG，
把每塊填色路徑接成一條 `svgRegion(color, d)` 加進 `PATTERNS`。
原則：**區塊少、形狀粗大**，沙子才填得漂亮。
